# 🚀 swiftkit

> REST client with intelligent caching

## ✨ Features

- Fast compilation (sub-second builds)
- Hot reload included
- Plugin system
- Zero config start

## 📦 Installation

```bash
npm install -g swiftkit
swiftkit init my-project
```
